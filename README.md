# VM汉化组网站状态检测
一个基于 UptimeRobot API 的在线状态面板

使用[site-status项目](https://github.com/imsyy/site-status)为基础开发

## 事先准备
- 需要到 [UptimeRobot](https://uptimerobot.com/dashboard) 添加站点监控，并在 `My Settings` 页面获取`API Key`

## 部署
### 安装依赖

```bash
pnpm install
```

### 开发
```bash
pnpm run dev
```

### 打包
```bash
pnpm run build
```
